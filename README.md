# Beschrijving

<p>Gemakelijkere manier om verschillende esp32-I2S games te spelen op een 64x32 matrix bord.</p>

# Gebruik

- server laten draaien op raspberry pi en verbind met esp32
- app instaleren en lokaal hosten
- via browser game selecteren om te spelen
